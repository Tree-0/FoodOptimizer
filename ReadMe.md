# Food Optimizer
## Overview
Find the optimal set of foods to minimize/maximize calories, hit nutrient requirements, and satisfy constraints.

In the future, hopefully support for pricing will be implemented, but it is very hard to find reliable price data on branded foods.

## Data
A combination of data from the Department of Agriculture FoodData Central's API and downloadable files were used.
**https://fdc.nal.usda.gov/**

Foundation Foods and Branded Foods datasets were downloaded from the following link:
**https://fdc.nal.usda.gov/download-datasets#bkmk-1**
